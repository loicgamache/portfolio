Site web de Scriptam Glassboards

Projet professionnel - Développement web

Description

Scriptam Glassboards est un site web où le but principal est de présenter les produits disponibles et faire une demande de prix. La partie la plus ardu a été le formulaire qui compte plusieurs versions selon les produits choisis. Ma tâche principale était de faire le HTML/CSS et de tout intégrer dans le gestionnaire de contenu Umbraco.

Logiciels et langages utilisés

  • Visual Studio
  • Figma
  • HTML
  • CSS
  • Javascript
  • Razor
  • Un peu de C#

Habiletés développées

Développement de formulaires complexes, configuration du gestionnaire de contenu Umbraco et utilisation du Razor.

Site web de recrutement Nmédia

Projet professionnel - Développement web

Description

Le site de recrutement Nmédia est un site axé sur le recrutement de main d'oeuvre qui contient des articles de blogues et les postes disponibles. Ce qui était le plus difficile dans ce projet était l'arrimage entre le designer et moi-même pour bien gérer le responsive du site web. Le site a été intégré dans le gestionnaire de contenu Altitude 3, fait par Nmédia. Le CMS ne permet pas l'utilisation des dernières balises HTML5. Le score de performance du site web pourrait donc être affecté.

Logiciels et langages utilisés

  • HTML
  • CSS
  • Javascript
  • Figma

Habiletés développées

Coordination entre designer et développeur, l'affichage mosaïque et l'implémentation d'un carrousel comportant des zones multiples.
